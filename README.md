# Angular Login Access

Stub with a generic Login Page for any front-end project

## Back-end

[node-js-es6-rest](https://github.com/marcosrachid/node-js-es6-rest)

## Installation

```bash
npm i -g yarn
yarn
```

## Execution

```bash
ng serve
```

or

```bash
yarn start
```

## ESLINT Validation

```bash
yarn lint
```
